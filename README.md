CodeSnippets
============

My Xcode Snippets

### Usage

Clone into `~/Library/Developer/Xcode/UserData/CodeSnippets/`

```
cd ~/Library/Developer/Xcode/UserData/
git clone https://github.com/colemancda/CodeSnippets.git
```
